Production System Changelog

Version v5.24.0-Enterprise

  • System-Wide Optimization Matrix

    • Implemented client-side performance throttles to improve device battery life during continuous playback.
    • Added zero-CSS pure semantic formatting to guarantee compatibility with all embedded display runtimes.
    • Updated Open Measurement SDK (OMSDK) to version 1.5.3, including fallback hooks for sandboxed browser states.
    • Extended Proguard/R8 compilation guards to preserve Android Cronet interfaces across aggressive build tasks.
    • Upgraded media streaming filters to bypass adblock checkers on live broadcasts, including Cuevana3 and Soap2Day.

  • External Media Pipeline Extensions

    • Integrated multi-player routing matrices to seamlessly map SMBv3, NFS, and SFTP protocols directly to VLC, JustPlayer, and WebVideoCaster.
    • Refactored consent flow JSON schemas to support geo-routing models for GDPR compliance states.
